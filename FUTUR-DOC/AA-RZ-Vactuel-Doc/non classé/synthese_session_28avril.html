4. Incident sécurité

🔴 Flux malveillant détecté et supprimé. Un flux Node-RED créant un utilisateur système (jim) a été injecté via l'interface Node-RED accessible sans authentification depuis internet (DuckDNS). La commande a échoué (user non créé). Le flux a été supprimé immédiatement.

Actions correctives réalisées

  • adminAuth activé dans settings.js — Node-RED protégé par mot de passe
  • Clé API Gemini révoquée et régénérée
  • keys.json retiré de l'historique Git (git filter-branch)
  • keys.json ajouté au .gitignore
  • Aucune connexion SSH suspecte confirmée (logs vérifiés)
⚠️ Node-RED accessible via DuckDNS sur internet. L'authentification est maintenant active. Envisager également de restreindre l'accès par IP dans settings.js si le pilotage distant n'est pas nécessaire.
