5. Contexte global Node-RED

RZ_GestionFlux_V3 alimente automatiquement le contexte global Node-RED à chaque trame reçue :

// Accès depuis n'importe quel flux Node-RED
const sys  = global.get('RZ_Sys');   // { dataContCpu: { SE: 42, _last: 42 }, ... }
const gyro = global.get('RZ_Gyro');  // { dataContGyro: { '*': [120,-45,300], ... } }
const cfgs = global.get('RZ_CfgS');  // { modeRZ: { '*': 2 }, ... }
✅La clé _last permet d'accéder rapidement à la dernière valeur sans connaître l'instance : global.get('RZ_Sys').dataContCpu._last
