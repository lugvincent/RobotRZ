Développement d'une 4ème version de robot de compagnie à bas cout. 
Il s'agit d'adapter la version précédente qui utilisait deux smartphone, un raspberry et 2 arduinos à un système simplifié :
Robot : méca structure à trois roues avec gestion capteurs et mouvement par arduino mega (associé à un uno dédié au déplacement) qui échange 
en liaison série avec un smartphone embarqué équipé de tasker pour le controle interne des  applis et capteurs.
Communication SE et Smart pilote (SP) via MQTT. Controle pilotage par programme Jscript + python (qd utilise joystick). 
Possibilité de controler sur tout sys exploit.
