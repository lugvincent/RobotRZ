# RobotZ_SE - Importation Tasker

1️⃣ Copier tous les fichiers sur le smartphone SE.
2️⃣ Ouvrir Tasker → long clic sur l’onglet “PROJETS” → “Importer”.
3️⃣ Sélectionner “Projet_RobotZ_SE.prj.xml”.
4️⃣ Tasker créera automatiquement la scène et les tâches.
5️⃣ Ajouter une autorisation :
   - Dans Android → Autoriser Tasker à lancer Termux.
6️⃣ Tester en appuyant sur “Démarrer l’application”.

Au démarrage, Tasker affichera la scène “RobotZ_SE”.
